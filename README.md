# WildCardWielder

Tool for learning words

### How to use

#### Build
```shell
$ mkdir build
$ cd build
$ cmake ..
$ make 
$ make install
```

#### Run
```shell
$ wcw -f /path/to/cards_file
```
Reverse word translation:
```shell
$ wcw -f /path/to/cards_file -r
```

#### Cards format
```
word1-translate1\n
word2-translate2\n
...
```
